# Wagtail CMS Setup Instructions

1. Install Python 3
2. Install Wagtail: pip install wagtail
3. Create a new Wagtail project:
   wagtail start mysite
4. Navigate into project folder:
   cd mysite
5. Run initial migrations:
   python manage.py migrate
6. Create a superuser for admin:
   python manage.py createsuperuser
7. Run development server:
   python manage.py runserver
8. Open browser at http://127.0.0.1:8000 to view site
