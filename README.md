# Wagtail CMS Demo Website

This is a sample Wagtail CMS demo website for GSoC.

## Features
- Homepage
- Blog page
- CMS editable content
- Built with Python Django and Wagtail

## Tech used
- Python
- Django
- Wagtail

## How to run
1. Install Python 3
2. Install dependencies: pip install wagtail django
3. Run development server: python manage.py runserver
4. Open browser at http://127.0.0.1:8000
